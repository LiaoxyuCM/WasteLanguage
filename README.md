# WasteLanguage
这是一个跑在HTML与JS上面的解释器，用于解释一个非常难用的自制语言。

[在线体验](https://waste.muah.top)

## 代码手册

| 符号 | 功能说明 |
|------|----------|
| #    | 初始化或重置指针为0。 |
| +    | 指针自加1。 |
| -    | 指针自减1。 |
| *    | 指针自乘2。 |
| /    | 指针自除2。 |
| %    | 直接打印指针。 |
| .    | 转成字符然后打印指针。 |
| :    | 打印换行。 |
| ?    | 暂停代码1毫秒。 |
| !    | 强制终止。 |
| @    | 清空输出。 |
| ,    | 打印后面的代码且不再执行后面的代码。 |
| \[\]   | 里面的代码执行2次。 |
| \(\)   | 50%概率执行里面的代码。 |
| \{\}   | 里面为注释。 |
| <    | 存档当前指针。 |
| >    | 读取存档并赋值给当前指针。 |
| ^    | 设置指针为0或1。 |
| &    | 设置断点。   |
